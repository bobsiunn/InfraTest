# 면접 과제: Degraded Compute Node

## 상황

컴퓨트 노드에 두 가지 문제가 보고되고 있습니다.

1. **스토리지**: 파일 읽기/쓰기 작업이
   비정상적으로 오래 걸리고, 일부 프로세스가 응답 없이 멈추는 현상이
   발생하고 있습니다.

   재현 예시:
   ```bash
   time dd if=/dev/zero of=/opt/compute/data/test bs=4k count=100 oflag=direct
   ```

2. **네트워크**: `apt`, `curl`, `wget` 등 외부 통신이 필요한 작업이
   시작 직후 수 초간 멈췄다가 진행되는 현상이 반복되고 있습니다.

## 목표

각 문제의 근본 원인을 찾아서, 재발하지 않도록 수정해주세요.
